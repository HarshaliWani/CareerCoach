# CareerCoach

🎯 Overview
AI-powered career coaching platform that helps students analyze performance and get actionable growth guidance.

🚀 Features
- Personalized career recommendations (faster decision-making)
- Student performance insights and guidance

🛠 Tech Stack
- React
- Node.js
- Python
- FastAPI

📈 Results
- Improved visibility into student performance trends
- Faster access to tailored career guidance

📋 Setup
1. Clone the repository.
2. Set up the backend:
	- Go to `backend/`
	- Install dependencies: `pip install -r requirements.txt`
	- Run the backend server.
3. Set up the frontend:
	- Go to `frontend/`
	- Install dependencies: `npm install`
	- Start the app: `npm run dev`
